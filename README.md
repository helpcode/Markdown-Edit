## 简介

`DBeditor` 是一款跨平台的`MarkDown`编辑器，支持主流的`Linux` / `Mac` / `Windows` 等操作系统，在各个系统上具有统一优秀的表现力。界面设计采用简洁扁平的UI风格，去除繁琐复杂多余的菜单，借鉴`Linux`上著名的`Vim`编辑器键盘快捷键命令操作风格，让用更加专注于文章博客的编写。
